# The App - DeFi Panda

## Main Problem

Finding comprehensive and reliable information about DeFi investments across web resources is
challenging and time-consuming. Users don't want to spend entire days conducting research and
analyzing data about ERC4626 vaults. DeFi protocols often prioritize positive public relations over
transparency, deliberately withholding critical data that might reveal vulnerabilities or
unfavorable metrics.

## Core Functionality - MVP Scope

- **Vault Tracking**: Monitor all DeFi products operating on the ERC4626 standard
- **Vault Directory**: Display a comprehensive list of tracked ERC4626 vaults with advanced features
  including sorting, searching, and filtering capabilities
- **Detailed Analytics**: Provide individual pages for each tracked ERC4626 vault featuring
  comprehensive insights such as:
  - Complete depositor registry
  - Historical deposit and withdrawal activity
  - Share price performance over time
  - Total Value Locked (TVL) trends
  - Vault holdings and investment positions
  - Average depositor position size
  - Underlying asset balance tracking
  - Direct links to original protocol websites

## Excluded from MVP Scope

- **No Transaction Capabilities**: The application serves purely analytical purposes - no
  depositing, withdrawing, or transaction execution
- **No Vault Management**: Users cannot add new ERC4626 vaults to the tracking system through the
  interface
- **No Backend Development**: GraphQL and REST API services are handled by a separate infrastructure
- **Web-Only Platform**: No mobile application development
- **No User Authentication**: No user accounts, wallet connections, or data sharing between users
- **No Data Export**: No functionality for exporting or downloading data

## Success Criteria

- Users can discover valuable insights for their technical and fundamental analysis
- Users report that the application provides helpful data for making informed investment decisions
- Users confirm that the platform reveals important information not readily available on original
  protocol websites
